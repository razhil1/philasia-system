Tips for Local Server Setup
External Drive Backup
Set BACKUP_DIR=D:\PhilAsia\Backups (Windows) or /mnt/backup (Linux) in your .env file.
Cloud Sync
Point your backup folder to a Google Drive or OneDrive sync folder for automatic cloud copies.
Scheduled Backups
Use Windows Task Scheduler or Linux cron to call the backup endpoint automatically each day.
